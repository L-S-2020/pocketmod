## Tools / Software
- [Kicad](https://www.kicad.org/) for pcb design
- [Onshape](https://www.onshape.com/) for CAD design
- [easyeda2kicad](https://github.com/uPesy/easyeda2kicad.py) a usefull tool to quickly get the symbols/footprints of LCSC / JLCPCB parts into kicad, can make mistakes tho, so double check the footprints before ordering
- [kicad-jlcpcb-tools](https://github.com/Bouni/kicad-jlcpcb-tools) a kicad plugin for generating correct production data for JLCPCB
